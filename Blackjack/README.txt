Blackjack Game:
    1. Deck Class
        - Choose number of decks
        - Shuffle method
    2. Card Class
        - Stores suit and value
        - Number card is its value
        - Face is 10
        - Ace can be 1 or 11
    3. Dealer Class
        - Deal cards to players
        - Draw cards for dealer
    4. Player Class
        - Gets cards from Dealer
        - Chooses whether to receive more cards from dealer
    5. Controller Class
        - Checks who won and loss the round
